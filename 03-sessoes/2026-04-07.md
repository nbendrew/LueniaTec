### Pipeline de automação de conteúdo no N8N mapeado
Fluxo completo para automatizar 100% da produção de posts no Instagram:

```
Schedule Trigger (todo dia às 8h)
  ↓
Claude API — gera tema do dia + roteiro + prompts
  ↓
Ideogram API — gera imagens (~R$0,06/imagem)
  ↓
Canva API — monta os slides automaticamente
  ↓
Telegram — envia preview para aprovação
  ↓
Buffer — agenda e posta no Instagram
```
